# 🌌 Cosmic Voyage

An interactive journey through the universe — from Earth's orbit to the edge of the observable cosmos.

**[✨ Start the journey](https://waffledolfi.github.io/cosmic-voyage/)** *(hosted with GitHub Pages)*

- 8 stages of travel: Earth orbit → the night sky's constellations → the planets → the stars → nebulae & black holes → galaxies & the whole universe
- 39 clickable objects, each with real NASA/ESA/ESO imagery, key stats and fun facts
- Ambient music generated live in your browser
- Track how much of the universe you've explored

Built as a single self-contained HTML file. Imagery courtesy of NASA, ESA/Hubble, and ESO.
